# Contributing

Thanks for contributing to Chaos Internet Simulator.

## Setup

1. Install Node.js 20+ and pnpm.
2. Install dependencies:

```bash
pnpm install
```

3. Start development:

```bash
pnpm dev
```

## Development flow

1. Create a branch.
2. Keep commits small and use conventional commits.
3. Run checks before opening a PR:

```bash
pnpm lint
pnpm test
pnpm build
```

## Pull requests

- Explain what changed and why.
- Add/adjust tests when behavior changes.
- Keep scope focused and avoid unrelated refactors.
