# Security Policy

## Supported Versions

Security updates are currently provided for the latest `main` branch.

## Reporting a Vulnerability

If you discover a security issue:

1. Do not open a public issue.
2. Contact maintainers privately with:
   - impact summary
   - reproduction steps
   - affected versions/commit
   - suggested fix if available

We will acknowledge reports as soon as possible and coordinate responsible disclosure.

## Scope Notes

- This tool is intended for local testing and development workflows.
- Never use this project to target systems without authorization.
